<!-- GETTING STARTED -->
## Getting Started

Voici tout ce qu'il faut faire pour lancer le projet :

### Prerequisites

Installe la dernière version de npm
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone le repo
   ```sh
   git clone https://github.com/TheoBourdel/MovieMaster.git
   ```
3. Installe NPM packages
   ```sh
   npm install
   ```
4. Lance le projet
   ```js
   npm start
   ```
